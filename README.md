# YAMAS

**Yoga Alignment and Movement Assistance System**
*Using Kinect Sensor for Real-Time Pose Assessment, Correction, and Guided Exercise*

---

## Overview

YAMAS is an experimental capstone project designed to assist users in improving posture alignment and flexibility through quantitative evaluation and real-time feedback. The system leverages a Microsoft Kinect sensor to capture and analyze body movement during yoga sessions, comparing user posture against predefined asanas and delivering corrective guidance.

---

## Key Features

**Real-Time Body Tracking**
Kinect-based desktop application that continuously tracks full-body movement during live yoga sessions.

**Rule-Based Pose Evaluation**
Compares user posture against a library of predefined yoga asanas using a structured rule-based evaluation engine.

**Quantitative Scoring System**
Measures joint angles, posture stability, and range of motion to produce an objective performance score for each pose.

**Real-Time Auditory Feedback**
Delivers spoken correction cues and guidance in the moment, without requiring the user to look away from their session.

**Flexibility Assessment Module**
Conducts an initial user assessment to evaluate baseline flexibility and recommend routines appropriate to the user's current range of motion.

**Intuitive User Interface**
Clean, user-friendly interface for routine selection, session configuration, and system interaction.
